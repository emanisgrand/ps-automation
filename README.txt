TO RUN THESE SCRIPTS

1. clone or zip files to a local directory
2. run your terminal 
3. navigate to '.\ps-automation' folder in your terminal 
4. run npm install 
5. run $ npm start
6. Manually close each instance of the browser
7. back at the terminal, run 'mocha data-tables.js' => close browser
8. at terminal, run 'node scrolltest.js' => close browser
9. at terminal, run 'node checkbox.js' => close browser
10. at terminal, run 'node dropdown.js' => close browser
